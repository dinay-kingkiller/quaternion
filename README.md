# Quaternion Rotation Notes

This repository contains a compact, implementation-oriented derivation of quaternion algebra with a focus on rotations in 3D space.

The material is written as structured notes rather than a formal textbook treatment, emphasizing definitions, identities, and computational forms useful for direct implementation (e.g., robotics).

Key conventions align with the tf2 transform system used in ROS, ensuring consistency with a common robotics practice.

## Document

The full notes are available here:

👉 https://dinay-kingkiller.github.io/quaternion/quaternion.pdf

## Topics covered

- Quaternion algebra (addition, multiplication, conjugation)
- Vector interpretation and pure quaternions
- Dot/cross product formulation
- Unit quaternions and quadrance
- Quaternion-based rotations
- Conversion to rotation matrices

## Notes

This is an evolving set of notes intended for clarity, correctness, and computational usability rather than completeness of formal exposition.
